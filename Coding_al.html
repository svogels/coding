Programming Basics

What is Programming?

Programming is telling a computer what to do. We give the computer a list of steps to follow. These steps are called code. We use languages like Scratch, Python, or Java to write code.

Block Coding

Block coding is a way to code by dragging and dropping blocks. It's like building with LEGOs. Each block is a piece of code. Scratch and Blockly are examples of block coding.

Which of these uses block coding?

Text-Based Programming

Text-based programming uses written words and symbols to code. It is more complex than block coding. Python, Java, and C# are examples of text-based languages.

Which of these is a text-based language?
